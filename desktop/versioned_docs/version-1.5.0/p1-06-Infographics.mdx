To read, understand, visualize and study the Bible through images and visual supports, the user should click on the **Infographics icon (1)** in the Side Navigation menu. 

<img src="/img/assets/infographics1.png"  width="1000px" alt="infographics1"/>
The user can select the language of the Bible, version, and Bible book from the drop-down menu (2).

In the drop-down menu, the user can select the language for viewing the Infographics **(3)**. 

If the book has Infographics, it will appear on the right pane **(4)**.  

The user can click on a card to enlarge the image in it. 

If Infographics are not available available for a specific book, the user will be notified with the message 'No Infographics available in English for 1 Chronicles'. 
Once the image is enlarged the user can rotate, zoom in or out, and navigate to the next and previous images by choosing the options listed below the image **(1)**. 

To exit the enlarged view, click the cross in the top right corner **(2)**.
